# Free Sewaa

Free Sewaa is a community sharing platform where people donate items for free to others who need them.

The goal of the platform is to reduce waste and support people who cannot afford essential items by connecting donors with recipients in a simple and accessible way.

---

## Purpose

Many useful items such as clothes, books, furniture, and electronics are thrown away while others struggle to afford basic necessities.

Free Sewaa helps solve this problem by creating a platform where community members can give away items for free and help others in need.

---

## Minimum Viable Product (MVP)

The first working version of Free Sewaa will include:

1. Users can post an item for donation  
2. Users can view a list of available items  
3. Users can request an item  
4. Users can contact the donor
