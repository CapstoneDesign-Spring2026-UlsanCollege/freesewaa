# Free Sewaa

Free Sewaa is a community-based donation and resource-sharing platform that connects people who want to give away usable items with those who need them.

---

## 📌 Project Overview

The goal of Free Sewaa is to reduce waste and support the community by making it easy to:

* Post items for donation
* Browse available items
* Request items
* Connect with donors for pickup

---

## 🎯 Target Users

* Students
* Low-income individuals and families
* Donors
* Anyone looking for reusable free items

## 📅 Development Process

This project is developed using a weekly sprint system:

* Each week includes planning, development, and review
* Work is tracked using GitHub Issues and Project Board
* Evidence is submitted through Weekly Sprint Packets
* GitHub is the single source of truth

---

## 🤝 Team Collaboration

* Roles rotate weekly (PM, Scribe, QA Lead, Demo Driver)
* All contributions are tracked via GitHub
* Pull Requests are used for all changes
* Each member provides contribution evidence
