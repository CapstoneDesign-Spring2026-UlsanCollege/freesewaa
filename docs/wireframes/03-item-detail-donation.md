# Wireframe Documentation for Item Detail and Donation Request Page

## Item Detail Page

### Header
- Logo: Top left corner
- Navigation Bar: Home, About, Contact, Donate

### Main Section
- Item Image: Large image of the item on the left side.
- Item Title: Bold and large font, centered above the item image.
- Item Description: Beneath the title, a detailed description of the item.
- Item Price: Displayed prominently beneath the description.
- Donation Button: Large button labeled "Donate Now" on the right side of the item image.

### Footer
- Links to social media accounts
- Contact information


## Donation Request Page

### Header
- Logo: Consistent with other pages.
- Navigation Bar: Home, About, Contact, Your Donations

### Main Section
- Form:
  - Item Title: Input field
  - Item Description: Text area
  - Place of Donation: Dropdown menu (e.g., Local Charity, Online, etc.)
  - Contact Information: Input fields for Name, Email, Phone Number
  - Submit Button: Large button to submit the donation request.

### Footer
- Links to social media accounts
- Contact information


### Notes
- Ensure accessibility features are included (e.g., alt text for images, proper labeling on forms).
