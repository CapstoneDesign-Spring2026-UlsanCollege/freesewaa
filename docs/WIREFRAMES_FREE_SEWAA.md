## Screen 1 - Entry / Home

Screen name:
Home / Browse Items

Purpose: What is this screen for?
This screen introduces Free Sewaa and helps users quickly browse available donation items.

Main user action:
Search or filter items, then open an item listing.

What appears on this screen:

  * title:
    Free Sewaa
  * short description:
    Community donation and resource-sharing platform for giving away usable items and helping people find what they need.
  * input/search/filter area:
    Search bar and category filter dropdown
  * main button:
    Browse Items
  * optional navigation:
    Post Item, Login, My Requests

What happens next:
The user searches or filters items, then clicks an item to see its details.

## Screen 2 - Core Task

Screen name:
Post Item

Purpose:
This screen lets a donor create a new item listing by entering the item details.

Main user action:
Fill out the form and submit the donation item.

What appears on this screen:

  * key content:
    Item title, category, condition, description, pickup location, contact method
  * form / interaction area:
    Text fields, dropdowns, and textarea for item details
  * main action button:
    Submit Item
  * validation or feedback note:
    Please complete all required fields before submitting.

What happens next:
After submitting, the item is saved and shown in the platform so others can browse and request it.

## Screen 3 - Result / Detail / Confirmation

Screen name:
Item Detail / Request Confirmation

Purpose:
This screen shows the selected item clearly and lets the user request it or see confirmation after requesting.

Main user action:
Review the item details and request the item or contact the donor.

What appears on this screen:

  * result / saved data / detail:
    Item name, category, condition, description, pickup location
  * confirmation or status message:
    Request submitted successfully
  * next action button:
    Contact Donor
  * fallback or error note if needed:
    Item no longer available

What happens next:
The user either contacts the donor to arrange pickup or returns to browse more items.
