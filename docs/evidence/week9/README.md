## How to Take Screenshots

### 1. Start the server
```bash
cd server
npm start
```

### 2. Open browser
Go to http://localhost:3000

### 3. Take screenshots
- Use ⌘+Shift+4 (Mac) or Win+Snip (Windows)
- Save to this folder

### 4. Verify CI
```bash
# Check GitHub Actions
# https://github.com/CapstoneDesign-Spring2026-UlsanCollege/Free_Sewaa/actions
```
