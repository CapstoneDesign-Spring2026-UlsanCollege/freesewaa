# Midterm Demo Guide

## Demo Flow (1 main user journey)

### User Story: Donating and Requesting an Item

1. **Landing Page** → Visit http://localhost:3000
2. **Sign Up** → Click "Sign Up", fill form
3. **Sign In** → Use credentials to login
4. **Donate Item** → Post a new item (jacket, books, etc.)
5. **Browse** → Browse available items
6. **Save Item** → Save an item to shortlist
7. **Request** → Request an item
8. **Messages** → Send message to donor
