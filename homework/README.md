# Homework

This folder is used to store homework-related files for the Free_Sewaa capstone project.

It helps keep homework materials separate from the main project source code, documentation, and other development files. Any assignments, homework submissions, or class-related tasks connected to the project can be added here.

## Purpose
- Keep homework files organized in one place
- Separate academic work from frontend and backend code
- Make it easier for team members and instructors to find homework materials

## Notes
- Only homework-related files should be stored in this folder
- Project source code should remain in the frontend or backend folders
- Official project documents should remain in the docs folder
